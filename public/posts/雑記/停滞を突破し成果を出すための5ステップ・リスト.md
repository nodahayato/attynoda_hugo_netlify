# 停滞を突破し成果を出すための5ステップ・リスト
## 1. 朝の「脳内排水（メンタル・クレンズ）」を行う 【所要時間：2分】
頭の中のノイズを取り除き、強制的に「行動モード」へ切り替えるためのステップです。

[ ] 頭の中の「気になっていること」を全て紙に書き出す

未完了のタスク、不安、アイデアなど、脳のリソースを奪っているものを全て外に出します。

[ ] その中で最も重いタスクを選び、最初の2分でできる「最小の一歩」を特定する

例：「企画書を書く」なら「ファイルを開いてタイトルだけ書く」、「ジムに行く」なら「ウェアに着替える」など。

[ ] その「最小の一歩」を今すぐ実行する

解説: 物理学と同じで、止まっている物体を動かすのが一番エネルギーを使います。一度動き出せば、慣性（勢い）で作業を続けられます。

## 2. 今日の「小さな勝利（スモール・ウィン）」を決める 【所要時間：2分】
壮大な目標に圧倒されて動けなくなるのを防ぎ、達成感を積み重ねます。

[ ] 今日達成可能な、具体的で小さな目標を1つ設定する

「世界を変える」ではなく「本を10ページ読む」「顧客に1件連絡する」など。

[ ] その目標を「具体的・測定可能・現実的・期限付き」で書き出す

解説: 巨大な成功も、日々の小さな行動の積み重ねでしかありません。毎日「勝った」という感覚を得ることで、自己効力感が高まります。

## 3. 不快なタスクには「5秒ルール」を発動する 【作業中随時】
「やりたくない」「怖い」と感じて先延ばししそうな瞬間に使います。

[ ] やらなきゃいけない事に対して「嫌だな」と感じたら、すぐに「5、4、3、2、1」とカウントダウンする

[ ] 「1」になった瞬間に、何も考えずに行動を開始する

解説: 脳は5秒以上考えると、やらなくていい言い訳（言い訳作り）を始めます。言い訳が生まれる前に体を動かして、脳のブレーキを無効化します。

## 4. 完璧主義を捨て、「70%」で世に出す（Ship Messy） 【判断基準】
準備不足を理由にチャンスを逃さないためのルールです。

[ ] 「まだ準備不足だ」「完璧じゃない」と感じたタイミングで、あえて人に見せる・公開する

目安は完成度70%。少し恥ずかしいくらいで丁度良いと心得る。

[ ] 「これはバージョン1です」と断って公開し、フィードバックを求める

解説: 完璧を目指して時間をかけるより、早く出してフィードバックを得て修正する方が、最終的な質もスピードも高まります。完璧は進歩の敵です。

## 5. センターピン（ワン・ドミノ）を見極める 【所要時間：3分】
あれもこれも手を出して分散するのを防ぎます。

[ ] 自分の目標を見直し、「これを倒せば、他のすべても倒れる（解決する）」という最重要課題を1つだけ特定する

例：著者の場合は「卓越したコンテンツを作ること」が、集客・採用・売上のすべてを解決するドミノでした。

[ ] その「1つのドミノ」を倒すことに、今日のリソースを集中させる

解説: 忙しいのに成果が出ないのは、重要でないドミノを倒すのに時間を使っているからです。本質的な課題1点に集中突破をかけます。
